# Wedding Prepare Preview

This public repository is a minimal static preview for mobile QA.

It intentionally contains only the files needed to run the H5 prototype:

- `index.html`
- `styles.css`
- `app.js`
- `executionChecklistCore.js`
- `config/execution-checklist-recommendations.js`

The private planning docs, issue drafts, validation scripts, Git history, and internal handoff materials are not included here.

## Preview Scope

Use this site to validate:

- Wedding map and node navigation
- Decision center interactions
- Execution checklist P0 flows
- Manual execution item CRUD
- Stage grouping, status toggle, stats, filters, and owner management
- Task, DecisionItem, and node recommendation joins
- Mobile layout and localStorage persistence
